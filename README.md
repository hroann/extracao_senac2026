# Contexto
Seu desafio será construir uma solução capaz de:

● Tratar problemas encontrados;
● Analisar as informações;
● Produzir conclusões e recomendações.

## O projeto deverá possuir as seguintes etapas:
# Parte 1 – Implementação do Pipeline de Dados
1. Extração de Dados, utilizar pelo menos duas fontes.
2. Limpeza dos Dados, aplicar pelo menos 3 técnicas.
3. Transformação dos Dados, criar pelo menos 5 novas colunas.
4. Análise Exploratória (EDA), aplicar pelo menos três análises.
5. Gerar visualizações, pelo menos 3 gráficos.
6. Criar um dashboard.
   
# Parte 2 – Apresentação dos Insights Gerados
Responder pelo menos 5 perguntas relevantes ao projeto.
Exemplos:Qual produto mais vendido? Qual categoria gera mais receita?
Quais clientes são VIP?
